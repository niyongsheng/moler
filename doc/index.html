Core Features

Explore Your Galaxy

Moler turns disk cleanup into a mission control experience. Every tool is a celestial instrument.

Scan & Clean
mo analyze / mo clean
Initiate a deep scan using mo analyze --json to find deletable files, then clean with real-time terminal streaming output and orbital radar animations.
Purge Projects
mo purge
Blast away project artifacts like node_modules, build caches, and derived data. PTY-driven terminal output with cancel support.
System Monitor
mo status --json
Live CPU, memory, network and disk stats polled every 3s. Rolling 30-sample sparkline charts and solar system orbit visualizations.
Disk Analysis
Treemap
Interactive treemap visualization of disk usage. See exactly what's consuming space at a glance — large files, cache directories, and more.
3D Jupiter
SceneKit
A fully interactive 3D Jupiter scene with 200-particle point cloud, Great Red Spot animation, 4 Galilean moons, and dark rings. Drag to rotate, scroll to zoom.
Dashboard Overview
At-a-glance
Mission control dashboard with stats cards, live CPU/memory gauges, disk capacity bar, network sparklines, and quick-action navigation to any module.
