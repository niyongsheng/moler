Core Features

Mission Modules

Five instrument panels plus a command dashboard. Each tool wraps a mo CLI command with a unique 3D scene and real-time streaming feedback.

System Scan
mo analyze / mo clean
Deep scan with mo analyze --json to find deletable files, review by category, then clean with real-time terminal streaming.
Project Purge
mo purge
Selectively remove build artifacts — Xcode DerivedData, node_modules, Gradle caches, SwiftPM, and more. Multi-select targets with terminal streaming.
System Optimize
mo optimize
Performance diagnosis and optimizations — DNS flush, Spotlight verify, Finder / QuickLook cache refresh. Dry-run preview with streaming task list and 3D Jupiter scene.
Disk Analysis
Treemap
Interactive treemap with breadcrumb navigation. Drill down nested blocks to find space hogs, backed by a 3D Earth scene.
App Uninstall
mo uninstall
Browse installed macOS apps, search and sort by name or size, preview trace files — App Bundle, Preferences, Caches, and more. Dry-run before you commit, with a 3D Mars scene.
Dashboard Overview
System Monitor
Live CPU, memory, network, and disk gauges. Health score, uptime, sparkline charts, and per-module stats at a glance.
